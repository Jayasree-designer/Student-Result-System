# Student Result System

## Description
This is a simple Java console application.
The program accepts student marks, validates the input,
calculates the percentage, and displays the grade.
It also allows entering marks for multiple students.

## Features
- Accepts marks between 0 and 100
- Validates invalid marks
- Calculates percentage
- Assigns grade (A, B, C, F)
- Allows multiple student entries

## Concepts Used
- if-else statements
- while loop
- break and continue
- methods (functions)
- Scanner class
- Console input and output

## Files Included
- StudentResultSystem.java
- output.png (program output screenshot)

## How to Run
1. Compile the program:
   javac StudentResultSystem.java

2. Run the program:
   java StudentResultSystem

## Output
The program displays:
- Student marks
- Percentage
- Grade  
Output screenshot is included in the repository.
